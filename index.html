Crystal Collector!

Instructions

The player will be given a random number from 18-120.
There are four crystals below by clicking on one of these crystals it will add a specific amount of points to the players score.
You win the game by collecting the correct number crystals equal to the original number given to you. Going any higher than the number will cause you to lose.
Each crystal's number is randomized each game.
Good Luck!
